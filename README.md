# Espacio-Cosmos
A simple page that explains what space is made of.

Tu página es una landing page temática del espacio, con efectos visuales modernos y animaciones elegantes. Está hecha usando HTML, CSS, TailwindCSS y JavaScript.

A continuación te explico cada parte en palabras simples:

## ⭐ 1. Configuración general

Declaras el documento HTML.

Defines que el idioma es español.

Agregas meta tags para que sea responsivo.

Le pones el título "El Espacio | Diseño Profesional Optimizado".

## ⭐ 2. Tailwind CSS personalizado

En el <script> inicial configuras Tailwind para usar:

La fuente Inter.

Colores personalizados como:

space-bg: negro espacial.

space-primary: texto claro.

space-secondary: gris suave.

space-accent: morado cósmico.

space-glass: efecto de vidrio.

Esto te permite usar una paleta profesional futurista.

## ⭐ 3. Estilos CSS avanzados

Aquí agregas efectos extra que Tailwind no trae por defecto:

Fondo dinámico que cambia solo en la sección de inicio.

Un overlay oscuro con blur para dar profundidad.

Animación brillante para los títulos.

Efecto “reveal” cuando haces scroll (aparecen suavemente).

Overlay de transición para animar los cambios cuando navegas entre seccion
